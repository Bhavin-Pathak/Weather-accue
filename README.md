
# Weather-accu

Weather App Using Openweather API-KEY With SWIFTUI


## Run Locally

Clone the project

```bash
   gitclone https://github.com/Bhavin-Pathak/Weather-accue.git
```

Go to the project directory

```bash
  cd Weather-accue
```

Open Xcode project

```bash
  Weather-accue.xcodeproj
```
